<Part id="db-part-3">

SQL được phân chia thành DDL (Data Definition Language: CREATE, DROP, ALTER) để định nghĩa cấu trúc và DML (Data Manipulation Language: SELECT, INSERT, UPDATE, DELETE) để quản lý dữ liệu.

:::highlight
Truy vấn INNER JOIN trả về các hàng từ cả hai bảng nếu có một sự khớp dữ liệu giữa các thuộc tính liên kết, giúp kết nối thông tin phân tán hiệu quả.
:::

</Part>
